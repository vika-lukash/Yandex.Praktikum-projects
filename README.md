# Проекты в Яндекс.Практикум
Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме, профессии "Аналитик данных".
| Название проекта | Описание  |  Навыки и инструменты |
| :------------------- | :----------------------------------------- |:---------------------------|
| Исследование надёжности заёмщиков — анализ банковских данных | На основе данных кредитного отдела банка исследовал влияние семейного положения и количества детей на факт погашения кредита в срок. Была получена информация о данных. Определены и обработаны пропуски. Заменены типы данных на соответствующие хранящимся данным. Удалены дубликаты.
